# Blog Application API

## Description

A basic blog application API, implemented with Nest.js, Postgres & TypeORM.

## Installation & Setup

```bash
$ npm install
```

Before running the app, create an `.env` file, filled with the data according to the `.env.example`.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```
